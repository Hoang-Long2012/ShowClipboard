# Show Clipboard content
Version: 1.0  
Author: Hoàng Long  
Repositories:  
https://github.com/Hoang-Long2012/ShowClipboard/  
This add-on has the function Show your clipboard content in a browseable window.

## Hotkey
- NVDA+Windows+X: Show your clipboard content in a browseable window.
- NVDA+Windows+Shift+X: Show your clipboard content in a browseable window with html format	.
